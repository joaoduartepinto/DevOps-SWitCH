## Create Tag
```shell script
$ git tag -a v1.2.0 -m "Version 1.2.0."
```

## Push Tag to remote repository
```shell script
$ git push origin v1.2.0
```

### Pull changes from remote

```
$ git pull origin <branch_name>
```

### Merge another branch with current one

```
$ git merge <branch_name>
```
